# FIMGen Web UI

A modern web interface for viewing, labeling, and editing datasets generated by the `fimgen` project.

## Features

- **Dataset Browser**: List and select `.jsonl` datasets from the `dataset/` directory.
- **Entry Viewer**: Inspect individual dataset examples with syntax highlighting.
- **Side-by-Side Comparison**: View Prompt and Completion side-by-side.
- **Interactive Labeling**: Quickly mark examples as Positive (üëç) or Negative (üëé).
- **In-Place Editing**: Edit Prompt and Completion text directly in the UI.
- **Dark Mode**: Optimized for long coding sessions.

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (v1.0 or later)

### Installation & Running

1.  **Navigate to the web directory**:
    ```bash
    cd web
    ```

2.  **Install Dependencies**:
    ```bash
    bun install
    ```

3.  **Start the Application**:
    ```bash
    bun run dev
    ```
    This will start both the backend (port 3001) and frontend (port 3000) concurrently.

4.  **Open in Browser**:
    Visit `http://localhost:3000` to use the application.

## Architecture

- **Backend**: Node.js + Express. Handles file I/O for reading and updating `.jsonl` datasets.
- **Frontend**: SolidJS + Vite. Provides a reactive and performant user interface.
- **Styling**: Vanilla CSS with a custom dark theme.
